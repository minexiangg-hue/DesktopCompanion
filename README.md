# 🎭 DesktopCompanion — 桌面人格陪伴宠物

> 每天早晨打开电脑，你的桌面小宠物都会给你一个"小惊喜"。

## 这是什么

一个**有灵魂的桌面陪伴组件**。不是效率工具，不是待办清单——是你桌面上一个活生生的电子伙伴。

它可以是：
- 🖼️ **风景大图推送器** — 每天早上给你一张震撼人心的风景
- 🤡 **搞笑梗图搬运工** — 让你笑着开始新的一天
- 📜 **历史今日播报员** — "1865年今天，某某某出生了……"
- 🎭 **人格模仿大师** — 用特朗普的语气说"Make America Great Again!"，用鲁迅的语气说"从来如此，便对么？"
- 📰 **新闻小能手** — "吃瓜啦吃瓜啦！某某明星结婚了！"
- 💬 **聊天搭子** — 双击它就能聊天，且永远在角色里
- 🤫 **猥琐发育的观察者** — 默默了解你，推送越来越懂你
- 💤 **会过日子** — 该睡睡该醒醒，不浪费你的 token

## 核心理念

> **"人格收集库"** — 不只一个萌宠，而是一个**人格剧团**。每天随机登台的演员不同，台词风格不同，给你的体验也不同。

三重人格切换逻辑（**并存**）：
1. **🎲 随机模式** — 每天自动换一个演员，保持新鲜感
2. **🎯 情境触发** — 检测到你熬夜 → 老妈人格催睡；周一早上 → 死侍人格吐槽
3. **👆 用户手动** — 打开人格库，今天想听谁翻牌谁

## 睡眠/清醒机制（省钱核心）

🟢 **清醒时段**（默认 08:00-22:00）：可聊天、可推送、可互动，token 按需消耗

🟣 **睡眠时段**（默认 22:00-08:00）：闭眼蜷缩，不说话，不调 LLM，零消耗

> 用户可自定义时间，临时点击睡着的宠物只会得到一句迷迷糊糊的 ZZZ。
